# Polaris Microservice Framework Project
Msf由前公司项目剥离出来，从入职时 SSH + Hessian 到 Dubbo SrpingBoot，从最开始的单一服务应用到分布式服务，耗费了很多的时光；为了保证服务的稳定安全监控等要求，也加入了各种优秀的组件。开源，一方面方便继续学习，跟上技术前沿的潮流；另一方面也算是提供了一种技术解决方案。
